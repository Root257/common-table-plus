# common-table-plus

基于Vue和Element-ui封装的通用Table列表组件，支持远程数据、搜索、导出、分页、表头筛选功能。
搜索功能：
    内置了基本的输入框、下拉框和时间选择框类型，支持传入自定义组件。
导出功能：
    支持自定义配置导出选项
表头筛选
    支持自定义筛选项，支持远程方法获取筛选数据


配置选项说明

|配置项|说明|详细|
|----|----|----|
|tableOptions|表格配置项|
|columns|列配置项|Object|
|formOptions|搜索条件Form表单配置项||
|exportOptions|导出配置项||
